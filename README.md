##### To run this application you need to execute the following commands in backend and frontend directories: 
      
      
### 1. Server directory
    - yarn install;
    - yarn knex:migrate;
    - yarn:knex seed;
    - yarn dev
### 2. Web directory
    - yarn install;
    - yarn start;


##### Automatically the application will open a tab on your browser on localhost:3000/.
